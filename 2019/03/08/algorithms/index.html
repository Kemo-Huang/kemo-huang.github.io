Combinational Number

C(n,m) = C(n-1,m-1) + C(n-1,m)

#include<iostream>
#include<vector>
using namespace std;
const int mod = 1000000007;
// 生成组合数表
vector<vector<long long>> C(int n, int m) {
    vector<vector<long long>> arr(n+1, vector<long long>(m+1, 0));
    for(int i = 0; i <= n; i++){
        arr[i][0] = 1;
    }
    for(int i = 1; i <= n; i++) {
        for(int j = 1; j <= m && j <= i; j++) {
            arr[i][j] = (arr[i-1][j] + arr[i-1][j-1]) % mod;
        }
    }
    return arr;
}

Sorting

Selection sort

  • 时间复杂度O(n^2)，不要使用

Bubble sort

  • 比插排慢，不要使用

Insertion sort

  • 越接近结果，速度越快
  • 只适合小范围的排序
  • 时间复杂度O(n^2)
  • 空间复杂度O(1)
  • 稳定

Heap sort

  • 适合内存小的环境，中小范围排序
  • 时间复杂度O(nlogn)
  • 空间复杂度O(1)
  • 不稳定

Quick sort

  • 比O(nlogn)要快，被用于java原始类型数组排序和C的qsort
  • 最坏情况O(n^2)
  • 空间复杂度O(logn)
  • 不稳定

Merge sort

  • 适用于数据被除主内存以外的地方获取
  • 时间复杂度O(nlogn)
  • 有很好的变种(Tim sort)
  • 空间复杂度O(n)
  • 比较难实现
  • 稳定
