树的基本概念和算法

平衡二叉树

自平衡二叉搜索树（Self-balancing binary search tree）又被称为AVL树（有别于AVL算法）

  • 它是一棵空树或它的左右两个子树的高度差的绝对值不超过1
  • 左右两个子树都是一棵平衡二叉树
  • 平衡二叉树的常用实现方法有红黑树、AVL、替罪羊树、Treap、伸展树等
  • 最小二叉平衡树的节点总数的公式如下 F(n)=F(n-1)+F(n-2)+1（Fibonacci数列），1是根节点，F(n-1)是左子树的节点数量，F(n-2)是右子树的节点数量。
  • h = O(logn)

插入

插在叶节点，根据情况旋转（最多一次）。

删除

目标节点无子节点，直接删。有一个子节点，删除后子节点顶替。有两个子节点，删除后用右子节点的最左叶节点顶替。最后根据情况旋转（可能两次）。

红黑树

  • 节点是红色或黑色。
  • 根是黑色。
  • 所有叶子都是黑色（叶子是NIL节点）。
  • 每个红色节点必须有两个黑色的子节点。
  • 从任一节点到其每个叶子的所有简单路径都包含相同数目的黑色节点。

优劣

在插入和删除上比AVL树快，但AVL树更适用于搜索。

void insert_case(node *n){
    if (n->parent == NULL) { // 根节点
        n->color = black;
    } else if (n->parent->color == BLACK) { // 父节点为黑
        return;
    } else if(uncle(n) != NULL && uncle(n)->color == RED) { // 叔父节点为红
        n->parent->color = BLACK;
        uncle(n)->color = BLACK;
        grandparent(n)->color = RED;
        insert_case(grandparent(n));
    } else {  // 父节点为红或叔父节点为黑
        if (n == n->parent->right && n->parent == grandparent(n)->left) { // 当前为右节点且父节点为左节点
            rotate_left(n->parent);
            n = n->left;
        } else if(n == n->parent->left && n->parent == grandparent(n)->right) { // 当前为左节点且父节点为右节点
            rotate_right(n->parent);
            n = n->right;
        }
        n->parent->color = BLACK;
        grandparent (n)->color = RED;
        if(n == n->parent->left && n->parent == grandparent(n)->left) { // 当前节点和父节点都为左节点
            rotate_right(grandparent(n));
        } else {  // 当前节点和父节点都为右节点
            rotate_left(grandparent(n));
        }
    }
}

完全二叉树

  • 若设二叉树的深度为h，1到h-1层的结点数都是满的，第h层所有的结点都连续集中在最左边。
  • 度为1的节点数为0或1
  • 叶子节点数为n/2或(n+1)/2
  • 转化为森林的个数为h或h-1
